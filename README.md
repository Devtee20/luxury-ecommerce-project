This is an interactive e-commerce web application built using HTML, CSS, and JavaScript, integrated with Firebase for backend services.

## 🚀 Features
- Product listing fetched from Firebase Firestore
- Search functionality for products by name
- View individual product details
- User authentication (sign up & login) using Firebase Authentication
- Add products to cart
- Increase and decrease product quantity
- Remove items from cart
- Cart data persistence using local storage
- Display total price of items in cart
- Checkout page with order summary
- Place order functionality
- Orders saved to Firebase Firestore with timestamps

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Firebase Authentication
- Firebase Firestore
- Local Storage
